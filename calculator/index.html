Уважаемые проверяющие, ниже приведена последовательность действий для проверки дополнительной функциональности.

Для действий с отрицательными числами, используйте кнопку +∕−. Сперва вводим число, затем нажимаем на кнопку.

Для извлечения квадратного корня: вводим число, нажимаем на √, получаем квадратный корень из числа.

Для возведения в степень: вводим число, нажимаем на ^, вводим степень.
